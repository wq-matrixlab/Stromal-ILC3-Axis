## C3⁺ Fibroblast–ILC3 Axis in Intestinal Inflammation

### Overview

This repository contains analysis code associated with a study investigating how intestinal fibroblast heterogeneity contributes to immune regulation during chronic gut inflammation.

The study focuses on a distinct C3⁺ fibroblast population enriched in the healthy intestine and reduced in inflammatory bowel disease. Through analysis of human and mouse intestinal datasets, the work explores how C3⁺ fibroblasts may act on group 3 innate lymphoid cells (ILC3s) via IGF1-mediated signaling to restrain inflammatory responses.

---

### Biological Focus

The central biological mechanism examined in this project is the regulation of ILC3 function by C3⁺ fibroblasts through IGF1-dependent signaling in the intestinal microenvironment.

Specifically, the analyses support the following concepts:

- C3⁺ fibroblasts represent a protective stromal population in the healthy intestine.
- This fibroblast population is reduced in the inflamed intestine of IBD patients.
- IGF1 acts as a key mediator of communication between C3⁺ fibroblasts and ILC3s.
- IGF1 signaling limits inflammatory CXCL10 production by ILC3s.
- Reduced control of CXCL10 may promote recruitment of plasmacytoid dendritic cells and amplify intestinal inflammation.
- The C3⁺ fibroblast–IGF1–ILC3 axis may represent an important stromal–immune regulatory pathway in IBD.

---

### Repository Scope

This repository provides the code used to examine gene expression patterns, stromal and immune cell states, and inflammatory signaling programs related to the C3⁺ fibroblast–IGF1–ILC3 axis.

The analyses include human IBD datasets, mouse intestinal datasets, and complementary transcriptomic analyses used to evaluate inflammatory responses and immune regulatory programs.

All datasets used in this repository are publicly available.

---

### Purpose

The purpose of this repository is to provide a transparent record of the computational analyses supporting the study. The code is intended for research reproducibility and for readers interested in exploring stromal–immune interactions in intestinal inflammation.

---

### Notes

This repository is intended as a companion resource to the associated manuscript. It does not contain private patient information or non-public clinical data.
